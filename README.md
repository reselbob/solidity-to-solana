# solidity-to-solana
A project that contains demonstration code for porting Solidity code to Solana

# Installation

## Install NodeJs

```
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash
```

```
source ~/.bashrc
```

```
nvm current
```

```
nvm install node
```

```
nvm use node
```

```
node -v
```
